Metrics

The following tables list the set of metrics included in the datasets.

MetricDescription

%VLOCvariation in the total number of lines of code that can be observed between two releases

%VJFvariation in the total number of java files that can be observed between two releases
%ConcrCpercentage of concrete classes that can be tested in each release
%VConcrCvariation in the total number of concrete classes that can be observed between two releases
%PLC percentage of line coverage
%VPLCvariation in the line coverage between two releases
%PBCpercentage of branch coverage
%VPBCvariation in the branch coverage between two releases
ACCxaverage value for the cyclomatic complexity
%VACCxvariation in the average value of cyclomatic complexity between releases
AMITCAverage of Method Invocations per Test Case
%VAMITCVariation in the average value of Method Invocations per Test Case between two releases
AICTCAverage of Instances Created per Test Case
%VAICTCVariation in the average value of instances Created per Test Case between two releases
AAFTCAverage of Assert and Fail instructions per Test Case
%VAAFTCVariation in the average value of Assert and Fail instructions per Test Case between two releases
ADACAverage of Disctinct Assert methods per Class

ADECAverage of Disctinct Exceptions per Class

#TLOCTotal of test LOC

ATCCAverage of Test Cases per Class
%VATCCVariation in Average of Test Cases per Class between two releases
